# Naeproblem

Small tools for everyday decisions.

Naeproblem is a lightweight directory homepage for independent practical tools. It uses a hub-and-spoke model: tools stay on their own domains and the homepage links out.

Project laws:
- Tool cards link out. Nothing is hosted on naeproblem.scot itself.
- No civic tools on the homepage.
- No sport or gambling-adjacent tools on the homepage.
- No shared nav injected into individual tool sites at this stage.
- No affiliate links on this page.
